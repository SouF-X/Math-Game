# Math Game

An early C++ console project built while learning programming fundamentals.

The game generates random math questions and lets the player choose the number of questions, difficulty level, and operation type.

## Features

- Choose number of questions from 1 to 10
- Difficulty levels: Easy, Medium, Hard, or Mixed
- Operation types: Addition, Subtraction, Multiplication, Division, or Mixed
- Random question generation
- Tracks right and wrong answers
- Shows pass/fail final result
- Allows replaying the game

## Concepts Practiced

- C++ basics
- Functions
- Enums
- Structs
- Loops and conditions
- Random numbers
- Console input/output
- Basic game flow
